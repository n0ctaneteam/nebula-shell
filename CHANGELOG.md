## [1.0.0] - 2026-07-04

### Added

- Core runtime with Application lifecycle management
- Reactive system with Observable, Property, Binding, and Computed
- Window system with Window, Panel, Popup, and Overlay support
- Widget system with Widget, Container, Box, Stack, Grid, and Overlay
- UI components: Label, Button, Entry, Separator, Spacer, Icon, Image
- Service architecture with Service base class and ServiceRegistry
- Configuration system with ConfigManager and hot reload
- Plugin system with PluginManager and lifecycle hooks
- IPC system with IPCServer and IPCClient
- Theme system with ThemeManager and CSS support
- Animation engine with Fade, Slide, and Scale animations
- Layer Shell integration for Wayland panels
- Hyprland backend support
- Event system for user interactions
- CLI interface with init, run, doctor, inspect, and dev commands
- Developer tools with hot reload, live CSS, and widget inspector
- Python bindings via GObject Introspection
- Example: basic_panel.py
